### Project Title
Explore US Bikeshare Data

### Description
This project uses Python to explore data related to bikeshare systems for three major U.S. cities - Chicago, New York City, and Washington. 

### Files used
bikeshare_final.py - final project
chicago.csv
new_york_city.csv
washington.csv

### Installation 
To run this project, you'll need Python 3, pandas and numpy installed on your computer. 

Python 3, NumPy, and pandas installed using Anaconda. hello

### Usage
To analyze the bikeshare data, run the bikeshare.py script in your terminal. This command will then launch in interactive mode where you can specify the city, month, and day for you analysis.

### Troubleshooting
Here are some common errors you may encounter while running the script:

Error: Invalid month input: Ensure you enter a valid month (e.g., "January" or "all") in proper capitalization.

Error: Dataset not found: Make sure you have downloaded the required dataset files (e.g., chicago.zip, new_york_city.zip, washington.zip) and placed them in the appropriate directory.
